# Vixel

Audio‑reactive 3D vector field, powered by Three.js and the Web Audio API.

## Quick start

- Use any static server and open the project root:
  - Node: `npx serve .` (or `npm i -g serve && serve .`)
  - Python: `python -m http.server 8000`
- Open the served URL and drag & drop an audio file to visualize.

## Highlights

- Real‑time FFT audio analysis
- Dynamic motion patterns and trails
- Color and size modulation
- Simple, keyboard/mouse‑friendly controls

## Tech

- Three.js + Web Audio API (runs in modern browsers)
